# 🚀 PokéDex React + TypeScript

Uma aplicação interativa e moderna desenvolvida em React para explorar os 151 Pokémon originais da região de Kanto, utilizando a [PokeAPI](https://pokeapi.co/).

---

## ✨ Funcionalidades (Features)

- [x] **Listagem Completa:** Exibição dos 151 Pokémon da 1ª Geração.
- [x] **Dark Mode:** Alternância entre temas claro e escuro para melhor conforto visual.
- [x] **Sistema de Favoritos:** Salve seus Pokémon preferidos (dados persistidos via LocalStorage).
- [x] **Filtros Inteligentes:** Busca por nome e filtragem por tipo (Fogo, Água, Erva, etc).
- [x] **UX Aprimorada:** - **Skeleton Screens:** Carregamento suave enquanto os dados são buscados.
- [x] **Scroll Restoration:** Salva a posição da página ao navegar.
- [x] **Botão Voltar ao Topo:** Navegação rápida em listas longas.

---

## 🛠️ Tecnologias Utilizadas

- [React](https://reactjs.org/) (Hooks, Memo, Router)
- [TypeScript](https://www.typescriptlang.org/) (Tipagem estática)
- [Vite](https://vitejs.dev/) (Build tool ultra rápida)
- [Axios](https://axios-http.com/) (Consumo de API)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) (Animações e Design Responsivo)
