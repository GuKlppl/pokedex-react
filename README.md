# 🚀 PokéDex React + TypeScript

Uma aplicação interativa e de alta performance desenvolvida em React para explorar o mundo Pokémon com uma interface moderna e imersiva.

---

## ✨ Funcionalidades

- [x] **Dashboard Moderno:** Layout horizontal em três colunas (Identidade, Status e Evoluções).
- [x] **Áudio Integrado:** Ouça o som original (Cry) de cada Pokémon diretamente na página de detalhes.
- [x] **Animações Fluidas:** Barras de status animadas com cores dinâmicas e transições suaves de 0.8s.
- [x] **Design Moderno:** ID do Pokémon em marca d'água gigante com tipagem geométrica e efeitos de opacidade.
- [x] **Sistema de Fraquezas:** Grade inteligente que calcula e exibe fraquezas elementares dinamicamente.
- [x] **Modo Shiny:** Alternância visual em tempo real para versões brilhantes dos Pokémon.
- [x] **Dark Mode & Responsividade:** Experiência otimizada para qualquer dispositivo e preferência de tema.
- [x] **Persistência:** Favoritos salvos via LocalStorage.

---

## 🛠️ Tecnologias & Engenharia

- **React 18 + TypeScript:** Tipagem rigorosa para maior segurança do código.
- **PokeAPI:** Integração avançada com consumo de dados de espécies, evolução e tipos.
- **Axios:** Gerenciamento eficiente de requisições assíncronas.
- **Modularização (Utils):** Lógica de negócios (cores, cálculos de fraqueza, formatação) isolada para facilitar manutenção.
- **CSS Avançado:** Uso de CSS Grid, Flexbox e Keyframe Animations.
